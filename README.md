# Super Store Sales Dashboard (Power BI)

This repository contains a Power BI project file (`.pbix`) for a comprehensive Super Store Sales Dashboard. The dashboard is designed to analyze and visualize sales performance, track key metrics, and derive insights from the Super Store dataset.
